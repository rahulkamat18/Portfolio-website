Featured Project

MeetSync

Real-Time Video Conferencing App

Think Zoom, but I built it from scratch. Real-time video/audio calls, screen sharing, chat — all running on WebRTC with Socket.io handling the signaling. The hardest part? Getting peer-to-peer connections to just work across different networks. But it does.

  • WebRTC peer-to-peer video & audio with low latency
  • Real-time chat + screen sharing
  • Secure auth with bcrypt & guest access via unique codes
  • Persistent chat history stored in MongoDB
